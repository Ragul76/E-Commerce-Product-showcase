🛒 E-Commerce Product Showcase

A simple and responsive E-Commerce Product Showcase Web Application built using HTML, CSS, and JavaScript. The application displays products in an organized layout with a focus on clean design, responsive user interface, and smooth browsing experience.

🔗 Live Demo:
https://ragul76.github.io/E-Commerce-Product-showcase/

✨ Features

📦 Display multiple products with images, names, and prices

🎨 Clean and responsive product grid layout

🖱️ Interactive UI with hover effects

📱 Mobile-friendly design for different screen sizes

⚡ Lightweight and fast-loading web interface

🛠️ Tech Stack

CSS3

JavaScript
